# yup-uniq

```
NAME:
   uniq - report or omit repeated lines

USAGE:
   uniq [OPTIONS] [INPUT [OUTPUT]]

      Filter adjacent matching lines from INPUT (or standard input),
      writing to OUTPUT (or standard output).

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --count, -c                    prefix lines by the number of occurrences (default: false)
   --repeated, -d                 only print duplicate lines, one for each group (default: false)
   --unique, -u                   only print unique lines (default: false)
   --ignore-case, -i              ignore differences in case when comparing (default: false)
   --skip-fields value, -f value  avoid comparing the first N fields (default: 0)
   --skip-chars value, -s value   avoid comparing the first N characters (default: 0)
   --help, -h                     show help
```
